# Rocketseat Full-Stack Journey

Este repositório contém todos os projetos, exercícios e desafios que desenvolvi durante a **Jornada Full-Stack** da Rocketseat. Aqui você encontrará desde os conceitos básicos de **HTML**, **CSS** e **JavaScript** até projetos mais avançados envolvendo **Node.js**, **React**, **TypeScript** e muito mais.

## 🚀 Sobre o Curso

A **Jornada Full-Stack** da Rocketseat é um programa completo que abrange tanto o desenvolvimento front-end quanto back-end, preparando os alunos para se tornarem desenvolvedores full-stack. O curso é dividido em módulos, começando com os fundamentos e evoluindo para tecnologias modernas e avançadas.

[🔗 Acessar Jornada Full-Stack da Rocketseat](https://app.rocketseat.com.br/journey/full-stack/contents)

## 📂 Estrutura do Repositório

O repositório está organizado em pastas, cada uma representando um módulo ou projeto específico. Abaixo está uma visão geral do que você encontrará aqui:

- **Fundamentos**: Projetos iniciais com HTML, CSS e JavaScript.
- **Front-end**: Desafios e aplicações utilizando React, TypeScript e outras tecnologias modernas.
- **Back-end**: Projetos envolvendo Node.js, APIs, bancos de dados e autenticação.
- **Desafios**: Soluções para os desafios propostos durante o curso.
- **Extras**: Projetos pessoais ou estudos complementares.

## 🛠️ Tecnologias Utilizadas

Ao longo da jornada, trabalhei com as seguintes tecnologias:

- **Front-end**: HTML, CSS, JavaScript, React, TypeScript, Tailwind CSS.
- **Back-end**: Node.js, Express, SQL, NoSQL, APIs REST.
- **Ferramentas**: Git, GitHub, VSCode, Figma.
- **Outros**: Conceitos de DevOps, deploy de aplicações, boas práticas de código.
